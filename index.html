How to Use

HTML


<div class="box">
    <div class="badge" role="status">New</div>
</div>

CSS


.box {
    position: relative;
    padding: 20px;
}

.badge {
    position: absolute;
    top: 0;
    right: 0;
    background: linear-gradient(45deg, #ff6b6b, #ff8e8e);
    color: white;
    padding: 0.5em 1em;
    border-radius: 20px;
    font-size: 0.875rem;
    font-weight: 600;
    animation: shimmer 2s infinite;
}

@keyframes shimmer {
    0% { opacity: 1; }
    50% { opacity: 0.7; }
    100% { opacity: 1; }
}
